Site update!
Posted by: MoraX (Admin / Developer) - Wednesday 25 July 2007, 15:15 Main
Country flags:
Well, the most noticable feature is probably the flags in front of usernames and clannames. Every user and clan which has selected a country, will get a flag in front of the name (where appropriate). If you don't know which country a user or clan is from, just hover over the flag itself, and the tooltip will show you which country is selected.

However, the country selection was added after the launch of the site. This means many users and clans have not selected their country yet. I would hereby like to ask everyone to do so, if you haven't already, thanks!

Ladders:
One of the biggest new features, is the introduction of ladders. QWL is now able to host ladders as well as cups! Although there are a few features not implemented yet, the ladders are ready to start.

Ladders work as follow:
A clan who just registered for a ladder, will be placed at the bottom. A clan can only challenge X clans above it (Where X is a variable which a ladder administrator can set). When a clan wins from a clan higher ranked, the winning clan will get the losing clans rank, and the losing clan drops one spot down.
So if clan A is on rank 15 and challenges clan B on rank 10 and wins, then clan A will get rank 10, and clan B will be placed on rank 9.

One of the things that still needs to be implemented is hibernating

Clanlist:
High on the request list, was a list of all clans registered at QWL. In the left menu block called "Quake Wars" you will now find a link "clanlist", which contains a list of all clans registered. It shows 25 clans per page, with an easy page switch box. Have fun!

Altered way of handling challenges:
The way a wararranger processes incoming challenges is altered a bit. Now, when pressing the "process" button, you will always go to 1 screen where all details of the challenge can be seen (date/time, opponent, cup/ladder, map). In the bottom of that screen you will find buttons for every option you have to further process the incoming challenge.
This way, you can always see all details, including an optional comment of the opponent before deciding what to do with the challenge

Timezones:
One very important note about timezones, because no-one seems to use them. The server we are running on, is located in The Netherlands, which means the server is runningin CET (GMT+1). If you live in a different timezone, you can select your timezone in your personal menu. When your timezone is different, all times are calculated to your timezone.
So if you live in GMT+2, and this newspost was written at 18:00 CET, then you will see 19:00 as time.
Although this is not very important for timestamps, it is for planning matches. When planning a match, your timezone is also taken into account!
Planning a match at 21:00 in GMT+6, will result in a date/time of 16:00 CET. So make sure your timezones are set correctly!
