Fix to Dual Core + FPS + Megatexture problem
Posted by: JB - Thursday 13 December 2007, 11:05 Main
Disclaimer: First, I am not claiming Im right about this being a fix in the first place. But it seems like it. I also dont know the correct technical terms for all this. And I am certainly not a SD engineer, or a game enginer designer.

However, this is my understanding of the situation & engine:

The problem
***********
Megatexture have level 0-4 (5 levels) of quality (layers of quality) that can be loaded into the game. And if you have dual core processor it is possible that ETQW engine doesnt perform like it should. The problem is seen when megatexture looks ugly.

Diagnostics tool:
TPS meter: Show TPS, Tiles Per Second

Put this bind in your config to esily toggle on and off the TPS meter.

Code:
bind KP_END "toggle com_showtps"

KP END = 1 button on keypad.
Press KP END to turn off / on.

Diagnostic if you have the problem (before fix):
***********
For me, levels 3 and 4 were loaded quickly to 100%, but levels 0-2 took very long time, perhaps needed to stand still for about 30 seconds in the same place for all levels to be loaded to 100%.


Setup your megatexture quality
***********
Begin with setting up megatexture to highest quality:

Code:
seta r_megaDrawMethod "0"

3 is default setting, which is the lowest quality
0 is highest quality
What I have seen, setting "3" is used even when using "High Settings" in menu. So this means, you need to change this cvar r_megaDrawMethod to 0 in console or in your config to get highest quality.

Fix to the problem
***********
Download the Windows XP patch for for dual cores (patch is for both AMD and Intel since version 4 of the patch what I understand). This XP patch isnt downloaded automatically. So it has to be downloaded manually.

If you dont have english version of your Windows XP, you need to download this patch in the same language that your XP use.

Filename:
WindowsXP-KB896256-v4-x86-ENU.exe

Download location

What I did was installing the file, rebooted PC, and then I had all levels at 100% all the time, even when running fast. Totally sick! In my personal opinion this Windows XP dual core patch should be included in ETQW installation.

The game also FELT with first impresstion that is had LESS jitter/stutter. That the game felt like its FPS was more fluid now. OMG! So good to feel the ETQW engine had more to deliver.
